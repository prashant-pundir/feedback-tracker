# Git Commands

## Clone Respository

```sh
$ git clone https://github.com/prashant-pundir/feedback-tracker.git
```

## Branches in local

```sh
$ git branch
```

## Local repository status

```sh
$ git status
```

## Local repository log

```sh
$ git log
```

## Create feature branch

```sh
$ git checkout -b feature-branch


## Add changes

```sh
$ git add GIT.md
$ git commit -m "Add git commands file"
